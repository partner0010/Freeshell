## 🚀 사용 방법

### E-book 생성
1. `/ebook` 페이지 접속
2. 주제, 콘텐츠 유형, 언어, 챕터 수 입력
3. "E-book 생성" 클릭
4. 생성 완료 후 가격 설정
5. Gumroad 또는 Etsy에 게시

### 블로그 자동화
1. `/blog` 페이지 접속
2. 주제, 콘텐츠 유형, 언어, 단어 수 입력
3. "블로그 포스트 생성" 클릭
4. 필요 시 다국어 번역
5. WordPress/Medium/Blogger에 게시

### 수익 확인
1. `/revenue` 페이지 접속
2. 플랫폼별 수익 확인
3. 기간별 필터링
4. 차트로 시각화된 수익 추이 확인

---

## 💡 참고사항

- API 키는 `.env` 파일에 `VITE_API_KEY`로 설정
- 백엔드 서버가 실행 중이어야 함
- 실제 플랫폼 연동은 각 플랫폼의 Access Token 필요
