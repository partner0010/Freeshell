- ✅ 메인 페이지: 올인원 콘텐츠 제작 플랫폼으로 리뉴얼 완료

  - 프로젝트 목록 표시
  - 통계 카드 (전체/완료/생성중/조회수)

## 🎯 현재 작동하는 기능

### 사용자 플로우
1. ✅ 로그인 → 올인원 스튜디오 대시보드 접근
2. ✅ "새 콘텐츠 생성" 클릭 → 생성 페이지 이동
3. ✅ 프롬프트 입력 → "콘텐츠 생성 시작" 클릭
4. ✅ 6단계 순차 실행 (스토리 → 캐릭터 → 장면 → 애니메이션 → 음성 → 렌더링)
5. ✅ 각 단계별 진행 상태 실시간 표시
6. ✅ 완료 후 프로젝트 상세 페이지로 이동
7. ✅ 프로젝트 목록에서 생성된 프로젝트 확인

### API 플로우
1. ✅ 프론트엔드 → `/api/allinone-studio/generate` 호출
2. ✅ 각 단계별 AI 역할 선택 및 프롬프트 생성
3. ✅ 무료 AI 서비스 호출 (Groq → Ollama → Fallback)
4. ✅ JSON 응답 파싱 및 반환
5. ✅ 프로젝트 저장 API 호출
6. ✅ 프로젝트 상세 조회
