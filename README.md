# 올인원 콘텐츠 AI

유튜브 숏폼 콘텐츠를 자동으로 생성하고 업로드하는 올인원 AI 솔루션입니다.

## 주요 기능

- 📝 다양한 콘텐츠 유형 지원 (이슈, 영화, 드라마, 개그, 일상대화, 감정 표현 등)
- 🤖 AI 기반 콘텐츠 자동 생성 (5가지 버전)
- 👀 실시간 미리보기 및 제작 이유 설명
- 🚀 자동 업로드 기능 (YouTube 등)

## 참고 사이트

### 유사 서비스
- [Pictory](https://pictory.ai) - AI 비디오 생성 플랫폼
- [InVideo](https://invideo.io) - 비디오 편집 및 생성 도구
- [Loomly](https://loomly.com) - 소셜 미디어 콘텐츠 관리

### 디자인 참고
- [Bityanolja](https://bityanolja.com) - 거래소 스타일 디자인

## 기술 스택

- **Frontend**: React + TypeScript + Vite + Tailwind CSS
- **Backend**: Node.js + Express (또는 Python FastAPI)
- **AI**: OpenAI API / Claude API
- **Video Processing**: FFmpeg

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```
